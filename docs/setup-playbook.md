# Setup-Playbook

Dieses Playbook richtet sich an die Person, die Kodi einmalig vorbereitet.

## Zielbild

Im Alltag sollen Seniorinnen und Senioren nur noch:

- Fernsehen oeffnen
- Mediatheken starten
- die Bibliothek oeffnen
- bei Bedarf in die Einstellung wechseln

## Einmalige Einrichtung

1. Kodi aktualisieren und `skin.kodi4seniors` installieren.
2. Einen PVR-Client konfigurieren und EPG-Daten pruefen.
3. Oeffentlich-rechtliche Mediathek-Add-ons installieren.
4. Den Skin als Standardskin aktivieren.
5. Optional unerwuenschte Menues oder Add-ons ausblenden.
6. Fernbedienungstasten fuer Home, Zurueck und OK pruefen.
7. Audioausgabe und Lautstaerkegrenzen testen.

## Abnahme

Vor der Uebergabe sollten diese Wege funktionieren:

- Startseite -> Fernsehen
- Startseite -> Mediatheken
- Startseite -> Bibliothek
- Startseite -> Einstellung

## Empfehlung

Wenn moeglich:

- nur notwendige Add-ons installiert lassen
- automatische Updates aktiviert lassen
- komplexe Bibliotheksmenues vermeiden
- eine grosse, kontrastreiche Ausgabeauflosung verwenden
